Get started

Five minutes to an AI ghost

01

Install Claude Code

npm install -g @anthropic-ai/claude-code
02

Clone & build

git clone https://github.com/thelastprompt/autoclaw.git
cd autoclaw && make run
03

API key + project

Menu bar → Settings. Add key. Point at a directory.

04

Wire up tools

Add MCP servers to ~/.claude/mcp.json.

05

Grant Accessibility. Press Fn. You have a ghost.
