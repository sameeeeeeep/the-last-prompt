Modes

Same ghost.
Different teeth.

Cycle with Option + X. Hard permission boundaries. Task ships code. Question can't touch a file. They never cross.

▸ Task

"Fix this. Ship this."

Full filesystem, shell, GitHub. Reads your code, finds the bug, patches it, runs the tests, opens the PR. You watch.

Architecture

Built different.
Literally.

Pure Swift. Apple Neural Engine. No Electron. No cloud relay. No telemetry. Your machine, your data, your code to audit.

Vision Pipeline

Apple VNFeaturePrint generates neural embeddings on-device. Zero API cost. Only meaningful screen changes trigger analysis.

let request = VNGenerateImageFeaturePrintRequest()
// On-device. Neural Engine. Zero cost.

Claude Code CLI

Tasks execute through Claude Code's CLI with streaming JSON. Every MCP server, skill, and permission — inherited automatically.

claude -p "fix the null ref" \
--output-format stream-json

MCP Protocol

Model Context Protocol gives Autoclaw reach. One JSON config, infinite integrations.

"mcpServers": {
"slack": { "command": "npx" }
}

Local-first

Screen captures, clipboard, and workflows never leave your laptop. Direct API calls to Anthropic. Nothing in between.

// ~/.autoclaw/
transcripts.db // local SQLite
world-model.md // your knowledge base

It watches your screen.
Nobody else does.

Mode-level isolation

+

Each mode defines explicit tool access. Task mode can write files and run shell. Question mode queries Granola and Sheets but can't open a file. Permissions are structural, not suggestions.

Local-first architecture

+

Everything runs on your machine. No cloud relay. No telemetry. Your clipboard, screen captures, and file contents never leave your laptop except as direct API calls to Anthropic.

Fully open source

+

Every line of Swift is on GitHub. Read the perception pipeline. Inspect the MCP wiring. Verify the permission model. Fork it, modify it, audit it.

Your Mac.
